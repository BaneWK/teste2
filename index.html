POP – Publicação de Projeto no GitHub

Objetivo

Demonstrar como subir um projeto no GitHub utilizando gravação de tela para fins de treinamento.

Ferramentas utilizadas

  • Git / GitHub
  • VS Code
  • Movavi Screen Recorder

Procedimento

  1. Abrir o terminal na pasta do projeto
  2. Executar git init
  3. Adicionar arquivos com git add .
  4. Criar commit com git commit -m "primeiro commit"
  5. Criar repositório no GitHub
  6. Conectar repositório remoto
  7. Executar git push
